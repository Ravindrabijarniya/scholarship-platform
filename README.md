# Scholarship Platform API

A Spring Boot backend for managing scholarships with authentication, scraping, and scheduling.

## Tech Stack

- Spring Boot 3
- PostgreSQL
- Redis
- JWT Authentication
- Docker
- Swagger (OpenAPI)

## Features

- User registration & login
- JWT authentication
- Refresh tokens
- Token blacklist
- Scholarship scraping
- Scheduled token cleanup
- Dockerized infrastructure

## Run with Docker

docker compose up --build

API:
http://localhost:8080

Swagger:
http://localhost:8080/swagger-ui/index.html
